𝐖𝐞 𝐧𝐨𝐰 𝐥𝐨𝐚𝐝 𝐋𝐃𝐫𝐚𝐰 𝐦𝐨𝐝𝐞𝐥𝐬 𝐦𝐮𝐜𝐡 𝐟𝐚𝐬𝐭𝐞𝐫 🚀 ... 𝐚𝐧𝐝 𝐢𝐧𝐜𝐫𝐞𝐦𝐞𝐧𝐭𝐚𝐥𝐥𝐲 📈. Incremental loading means that CAD models are loaded 𝘱𝘢𝘳𝘵 𝘣𝘺 𝘱𝘢𝘳𝘵. With this approach our users get 𝘣𝘦𝘵𝘵𝘦𝘳 𝘷𝘪𝘴𝘶𝘢𝘭 𝘧𝘦𝘦𝘥𝘣𝘢𝘤𝘬 on the loading state of the CAD models. This feedback greatly improves the 𝘶𝘴𝘦𝘳 𝘦𝘹𝘱𝘦𝘳𝘪𝘦𝘯𝘤𝘦.

𝐅𝐨𝐫 𝐭𝐡𝐞 𝐥𝐨𝐚𝐝𝐢𝐧𝐠 𝐨𝐟 𝒑𝒂𝒓𝒕𝒔 we still use the standard 𝘓𝘋𝘳𝘢𝘸𝘓𝘰𝘢𝘥𝘦𝘳 from 𝘛𝘩𝘳𝘦𝘦.𝘫𝘴. The standard loader turns LDraw part files into corresponding 3D objects with faces, edges, and their colors. To speed up the loading process, we disabled the smoothing of face normals (which is enabled by default).

𝐅𝐨𝐫 𝐭𝐡𝐞 𝐥𝐨𝐚𝐝𝐢𝐧𝐠 𝐨𝐟 𝒂𝒔𝒔𝒆𝒎𝒃𝒍𝒊𝒆𝒔 we use a 𝘤𝘶𝘴𝘵𝘰𝘮 LDraw loader instead.  The custom loader first parses the LDraw assembly files. Then, the loader detects references to LDraw part files inside the assemblies. Finally, the loader uses the standard 𝘓𝘋𝘳𝘢𝘸𝘓𝘰𝘢𝘥𝘦𝘳 from 𝘛𝘩𝘳𝘦𝘦.𝘫𝘴 for loading the parts.
