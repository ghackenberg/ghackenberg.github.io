In our #simulation courses we learn to use different techniques for estimating #system performance characteristics, e.g. product lead times or machine utilization.

One such technique is Monte-Carlo-Simulation, which works well with probabilistic system models. Probabilistic models typically encode some degree of #uncertainty, e.g. about machine breakdowns and repair times. During simulation, one can roll the dice to resolve this uncertainty (hence, the name Monte-Carlo-Simulation). Enough repetitions finally will yield stable predictions of the desired system performance characteristics.

A basic #application of Monte-Carlo-Simulation is estimating the number PI. In this application, thousands of random points are generated within the unit square. Then, the number PI can be estimated from the total number of points (i.e. green and red points in the video) versus the number of points, that lie within the quarter circle (i.e. only red points in the video).
