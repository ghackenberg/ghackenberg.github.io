Some time ago we started working on 𝐂𝐨𝐦𝐩𝐮𝐭𝐚𝐭𝐢𝐨𝐧𝐚𝐥 𝐅𝐥𝐮𝐢𝐝 𝐃𝐲𝐧𝐚𝐦𝐢𝐜𝐬 (𝐂𝐅𝐃) 𝐟𝐨𝐫 𝐋𝐃𝐫𝐚𝐰™ in CADdrive using 𝐎𝐩𝐞𝐧𝐅𝐎𝐀𝐌. The first results look quite promising!

Note that OpenFOAM does not support the LDraw data format directly. So to make this work, we ended up converting our LDraw models into the OBJ Wavefront format using LeoCAD, before loading the models into OpenFOAM and doing the remaining CFD work.

Next, we plan to integrate this functionality into the 𝐂𝐀𝐃𝐝𝐫𝐢𝐯𝐞 𝐖𝐞𝐛 𝐀𝐏𝐈 and the 𝐂𝐀𝐃𝐝𝐫𝐢𝐯𝐞 𝐖𝐞𝐛 𝐆𝐔𝐈. So soon, you can run basic CFD experiments on your LDraw models with the click of a button!
