description: "At CADdrive we are currently working on FEM simulation of arbitrary LEGO structures. We create our digital LEGO models with the free and open source software LeoCAD, which builds upon the open LDraw data format and LEGO parts library. To enable FEM simulation, we developed a basic algorithm and Python program that converts the LDraw models into CodeAster models. Then, we use CodeAster to perform the actual FEM simulation and return the force and displacement data. The visualization of the FEM simulation results is finally created using Panda3D, a free and open source 3D engine for Python programmers."
