<p>
			MaCon is implemented using <a href="http://docs.oracle.com/javase/8/index.htm" target="_blank">Java</a>, <a href="http://docs.oracle.com/javase/8/javase-clienttechnologies.htm" target="_blank">JavaFX</a>, and <a href="http://jbullet.advel.cz/" target="_blank">JBullet</a>.
			For development, we use <a href="https://maven.apache.org/" target="_blank">Apache Maven</a> and <a href="https://eclipse.org/" target="_blank">Eclipse</a>, but any other Java IDE can be used as well.
			Internally, MaCon comprises five modules: The core, the model, the syntax checker, the semantics checker (or simulation engine), and the workbench.
			The workbench includes the previous four modules and adds a graphical user interface.
